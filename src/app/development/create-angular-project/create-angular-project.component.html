Now that we have everything set up, let's create a new Angular project.

  1. Create a Project:
    • In your terminal, navigate to the folder where you want to create your project. For example, if you want to create it on your desktop, you can use the command:
      • bash 
      cd Desktop
    • Run the following command to create a new Angular project:
      • bash 
      ng new my-angular-app
    • Replace "my-angular-app" with your desired project name. The CLI will ask you a few questions, such as whether to include Angular routing and which stylesheet format you'd like to use (CSS, SCSS, etc.). Choose your preferences or stick with the defaults.
  2. Navigate to the Project Folder:
    • After the project is created, navigate into the project folder using:
      • bash 
      cd my-angular-app
  3. Start the Development Server:
    • To start the development server, run:
      • bash 
      ng serve
      OR
      bash 
      ng serve --open
    • The server will start, and you can view your project in a web browser at http://localhost:4200. This is your live preview, where any changes you make will be updated automatically.

Congratulations!

You've successfully set up your development environment and created your first Angular project. In the next sections, we'll explore how to build components, style your application, and more. Stay tuned and happy coding!
