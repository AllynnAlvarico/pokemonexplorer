The Angular CLI (Command Line Interface) is a tool that helps you create and manage Angular projects. It simplifies many tasks, like setting up a new project and running a development server.

  1. Install Angular CLI:
    • In your terminal or command prompt, type the following command and press Enter:
      • bash 
      npm install -g @angular/cli
    • The -g flag stands for "global," meaning the Angular CLI will be available for any project on your computer.
  2. Verify the Installation:
    • To ensure Angular CLI is installed correctly, type the following command and press Enter:
      • bash 
      ng version
    • You should see information about the Angular CLI version, Angular version, and other details.
