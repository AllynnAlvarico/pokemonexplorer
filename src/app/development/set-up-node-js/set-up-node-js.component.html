Before installing Angular, you will need to set up Node.js and npm (Node Package Manager). Node.js is a JavaScript runtime that allows you to run JavaScript on the serve, while npm is a tool that helps you manage packages (libraries) that we need for our projects.

  1. Download Node.js:
    • Go to the Node.js website
    • Download the LTS (Long Term Support) version, which is recommended for most users.
    • Follow the installation instructions for your operating system (Windows, macOS, or Linux).
  2. Verify the Installation:
    • After installing, open your terminal, command prompt or powershell.
    • Type "node -v" and press Enter. You should see the version number of Node.js you installed.
      • bash 
      node -v
    • Type "npm -v" and press Enter. This will show the version number of npm.
      • bash 
      npm -v
